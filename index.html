ConGen Quick Start

  • Adding Content: Click "+ Add Page" to start a new document section. Click "+ Add Block" to append Text, Images, Spacers, or Row Containers to that page.
  • Row Containers: Row Blocks act as flexible grids. Switch internal blocks to "Auto Space" width and they will perfectly divide the horizontal space. Set an internal block to "Half Width" and the siblings will squish around it. (Max 4 blocks per row).
  • Reordering: Grab the "☰" icon on any block to drag and drop it anywhere, even into other Row Containers.
  • Exporting: Hit "Export Final Documentation" to download your complete workspace as a fully functional, self-contained HTML file.
📚 Looking for more detail?
The developer team has published an interactive ConGen User Manual built inside the generator itself. Look in your root directory for ConGen_User_Manual.html and load it using the "Need to edit an old file?" picker!
